# Widgets

Übersicht über alle verfügbaren Widgets in ioBroker.aura.

## Standard-Widgets

| Widget | Beschreibung | Status |
| --- | --- | --- |
| [Schalter](./schalter) | Boolean-Schaltflächen mit mehreren Layouts | Dokumentiert |
| Lampe | Licht mit Helligkeit, Farbe und Farbtemperatur | _geplant_ |
| Liste | Statische oder dynamische Listenansichten | _geplant_ |
| Universal | Frei konfigurierbare Zellen | _geplant_ |

> Diese Übersicht wächst mit der Doku. Pro Widget gibt es eine eigene Seite mit Layouts, Einstellungen und Screenshots.
